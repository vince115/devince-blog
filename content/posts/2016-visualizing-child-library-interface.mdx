隨著兒童數位閱讀需求增加，傳統以文字導向的搜尋介面不再適用。  
《兒童數位圖書館資訊視覺化網站設計─以圓夢繪本資料庫為例》這篇研究，提出以 **資訊視覺化（Information Visualization）** 與 **使用者參與設計（Participatory Design）** 為核心的網站開發模式，打造學齡前兒童專用的搜尋體驗。

技術上，系統採用 **MVC 架構**：  
後端以 **Node.js + PHP Slim Framework + MySQL** 實作 **RESTful API**，以 JSON 格式傳遞繪本資料；前端則運用 **HTML5、CSS3、JavaScript、jQuery** 與 **D3.js**，將資料視覺化為互動式圖形。

研究設計了三種介面模式：  
1. **主題地圖模式** — 以生活場景（學校、家、動物園等）作為地圖節點，結合 SVG 動畫與滑鼠互動，模擬探索遊戲。

3. **動態節點介面模式** — 採用樹狀節點與群集動畫（Cluster Model），呈現知識關聯與層級。

前端透過 **D3.js 的資料綁定與 transition 動畫** 實現節點展開效果，結合 **SVG 向量繪圖** 以提升互動流暢度與可擴充性。  
實驗結果顯示，主題地圖模式在趣味性與搜尋效率上表現最佳，證實資訊視覺化能顯著降低兒童使用障礙，並提升網站重訪率。
