## Temple Oracle — 數位宗廟抽籤體驗

**專案類型：** 文化儀式互動網站  
**技術架構：** Next.js 14 + TypeScript + TailwindCSS  
**核心特色：** 直排籤詩排版（writing-mode: vertical-rl）  
**資料來源：** 結構化 JSON 籤詩資料  

---

## 專案理念

Temple Oracle 是一個以東方宗廟抽籤儀式為靈感所打造的數位體驗專案。

本專案並非單純顯示籤詩結果，而是試圖重現「抽籤的儀式感」——  
從選擇神祇、抽籤流程、到閱讀直排籤文與判詞，  
讓使用者在數位介面中感受到接近真實宗廟氛圍的體驗。

---

## 設計思考

整體設計強調：

- 極簡留白
- 儀式氛圍
- 文化質感
- 傳統排版比例

籤詩採用 `writing-mode: vertical-rl` 呈現，  
模擬傳統寺廟籤紙的直排閱讀方式。  
框線、留白與字距皆依照實體籤紙比例進行微調。

> 科技不是取代文化，  
> 而是延伸文化的呈現方式。

---

## 互動流程設計

使用者體驗流程包含：

1. 選擇神祇（觀音、月老、城隍）
2. 進行數位抽籤互動
3. 顯示籤詩等級與籤文
4. 閱讀詳細運勢分類（婚姻、財運、健康、行人、訴訟等）

流程設計偏向沉浸式體驗，而非遊戲化效果。

---

## 技術架構

- 模組化 JSON 籤詩資料管理
- 元件化 UI 架構
- 支援桌面與行動裝置 RWD
- 深色 / 淺色主題模式
- 清楚分離抽籤邏輯與顯示層

未來可擴充更多神祇與籤詩資料，而不需重構核心系統。

---

## 專案反思

Temple Oracle 是一次「文化介面工程」的實驗。

它嘗試證明：  
傳統儀式與現代前端技術並不衝突，  
只要設計得當，文化與科技可以共存，  
甚至互相強化。
