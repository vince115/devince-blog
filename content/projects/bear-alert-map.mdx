## 🧸 專案介紹

熊出沒警報地圖是一個以 **地理資訊視覺化** 為主題的前端專案，  
透過 Leaflet.js 實現互動式地圖，讓使用者能即時查看熊出沒的地點與警報區域。

系統採用 **Next.js 14 + TypeScript** 開發，並使用 **Leaflet** 搭配地圖圖層標記（marker）與事件描述。  
資料可由後端 API 或靜態 JSON 載入，支援桌面與行動裝置的 RWD 響應式設計。

---

### 🔍 主要特色
- 🗺️ **互動地圖顯示**：以 Leaflet.js 呈現熊出沒警報區域  
- ⚠️ **警報分級與地標標示**：不同等級以顏色標示  
- 📍 **點擊查看事件細節**：地點、時間、描述等資訊  
- ⚡ **Next.js 14 + TypeScript 架構**：支援靜態與動態渲染  
- 📱 **RWD 響應式設計**：行動裝置與桌面皆可流暢操作
